Factor A - Proyecto base PWA (actualizado)
Incluye:
- UI LED azul y responsive (PC y móvil)
- Configuración centralizada con tema y fuentes
- Funcionalidad implementada para:
  - Escuela: materias con horarios, TPs (adjuntos) y pruebas
  - Médico: turnos asignados y turnos por sacar
- Persistencia en LocalStorage
Cómo usar:
1. Colocar todos los archivos en una carpeta y abrir index.html en el navegador.
2. Para probar service worker (installable) usá un servidor local:
   python -m http.server 8000
   y abrir http://localhost:8000
